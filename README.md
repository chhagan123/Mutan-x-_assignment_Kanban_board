📝 Kanban Task Management Board

A Kanban-style task management application built with React, Tailwind CSS, and DnD Kit.
This project helps users manage tasks efficiently with drag-and-drop functionality, 
column-based workflow, and features like add, edit, delete (with confirmation modal), undo/redo, and theme switching

🚀 Features

✅ Add, Edit, and Delete tasks

✅ Add and Delete columns with confirmation modal

✅ Drag and Drop tasks between columns

✅ Undo/Redo functionality

✅ Light/Dark theme toggle

✅ Search tasks easily

✅ Responsive design with Tailwind CSS

 🛠️ Tech Stack

State Management: React useState, useEffect

Drag and Drop: DnD Kit

Icons: Lucide React

Deployment: Netlify
